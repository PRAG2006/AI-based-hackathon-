# ClassVoice AI

ClassVoice AI is an AI-powered classroom participation analysis system designed to identify participation inequality, detect hidden engagement barriers, and support inclusive learning environments.

---

## Features

- Classroom participation analytics
- AI-generated engagement insights
- Silent learner detection
- Participation heatmaps
- Interactive dashboard
- AI classroom assistant
- Real-time activity monitoring
- Predictive engagement analysis
